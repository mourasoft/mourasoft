## 🚀 Featured Project

### [🤖 LLM Platform on Kubernetes](https://github.com/mourasoft/llm-platform-k8s)

> Production-grade AI serving platform built from scratch — multi-tenant, GitOps-driven, zero-trust security model, automated MLOps pipeline.

```
┌─────────────────────────────────────────────────────────────┐
│  LLM Serving    │  Ollama · TinyLlama · KServe              │
│  Autoscaling    │  KEDA scales on ollama_active_requests     │
│  GitOps         │  ArgoCD app-of-apps · selfHeal < 30s      │
│  Observability  │  Prometheus · Grafana · SLOs · Alerts     │
│  Security       │  OPA Gatekeeper · Falco · Trivy · NetPol  │
│  MLOps          │  Tekton · ArgoCD Rollouts · MinIO          │
└─────────────────────────────────────────────────────────────┘
```

**What it demonstrates:**

- **Multi-tenant isolation** — ResourceQuota + NetworkPolicy across 4 namespaces
- **Queue-based autoscaling** — KEDA reacts to `ollama_active_requests` in real time
- **GitOps self-healing** — ArgoCD reverts any manual drift in under 30 seconds
- **Zero-trust by default** — OPA blocks root containers; Falco detects runtime threats
- **Full MLOps loop** — Tekton evaluates → canary deploy → auto-rollback on failure
